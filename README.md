# 🇳🇬 AYO – A Pidgin Health Language Assistant

AYO is a Gradio-based prototype for culturally aligned healthcare communication in Nigerian Pidgin.

## Features
- Nigerian green-and-white UI
- Prompted health-information assistant in Pidgin
- Ready to connect to a fine-tuned LoRA adapter
- Local demo for thesis presentation

## Run locally
```bash
pip install -r requirements.txt
python app_gradio.py
